What is Style Manager?

Style Manager is a powerful Google Docs add-on that lets you create, customize, and apply advanced text styles effortlessly. Designed for productivity and style consistency, it's the ultimate tool for professionals, educators, writers, and teams.

Features

  • Toggle text formatting: Bold, Italic, Underline, Strikethrough
  • Control paragraph indents: before text, after text, and first-line
  • Capitalization options: UPPERCASE, Capitalize Words, lowercase, Sentence case
  • Text alignment: Left, Right, Center, Justify
  • Customize line spacing, font color, highlight color
  • Select from any font family and size
  • Live preview styles in the sidebar
  • Save and edit your styles for global or document-specific use
  • Organize styles into custom groups
  • Extract styles from selected text to reuse and update
